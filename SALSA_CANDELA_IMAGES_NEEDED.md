## Image Specifications

### Format
- **Preferred:** PNG for screenshots, WebP for optimized web delivery
- **Acceptable:** JPG for photos (party images)
- **Avoid:** BMP, TIFF (too large)

### Size Guidelines
- **Featured/Hero images:** 1200-1600px wide
- **Full-width screenshots:** 1200px wide minimum
- **Side-by-side images:** 600px wide each
- **Mobile screenshots:** 375-414px wide (iPhone size)
- **Icons/small images:** 400-600px wide

### Optimization
- Compress images for web (use TinyPNG or similar)
- Target: <500KB per image for fast loading
- Use descriptive filenames (already done above)
- Add alt text in HTML (already done in blog post)

### Tools
- **Screenshots:** macOS Screenshot (Cmd+Shift+4) or Windows Snipping Tool
- **Browser DevTools:** For mobile responsive screenshots
- **Design/Graphics:** Figma, Canva, or Photoshop for comparison tables
- **Code Screenshots:** VS Code with theme, zoom to 150-200% for readability
- **Optimization:** TinyPNG, ImageOptim, or Squoosh
